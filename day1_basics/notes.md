# Dia 1 - Fundamentos de Python

# Tipos de Dados
- Inteiros: `int`
- Decimais: `float`
- Texto: `str`
- Booleanos: `True`, `False`

## Operadores
- Aritméticos: `+`, `-`, `/`, `*`, `//`, `%`, `**`
- Comparação: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Lógicos: `and`, `or`, `not`

## Manipulação de Strings
- Acessar letras: `texto[0]`
- Fatiamento: `texto[1:4]`
- Métodos: `.upper()`, `.lower()`, `.strip()`, `.replace()`, `.split()`

## Coleções
- Listas: `[]`
- Tuplas: `()`
- Dicionários: `{chave: valor}`

## Entrada e Saída
- `input()`
- `print(f"Olá, {nome}")`
